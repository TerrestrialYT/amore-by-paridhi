# 🍰 Amore By Paridhi - Premium Bakery Landing Page

A bespoke, high-converting, single-page landing page built for "Amore By Paridhi." This project focuses on luxury UI/UX, utilizing a warm, inviting color palette and buttery-smooth scroll animations.

### ✨ Features
* **Aesthetic Design:** Soft warm cream backgrounds (`#fdfbf7`), earthy espresso brown typography (`#4a332a`), and watercolor blush pink hover states (`#f2c9cf`).
* **Parallax Hero Section:** A full-width, fixed-background hero layout with an overlapping content box for a modern, dimensional feel.
* **Scroll Animations:** Powered by [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) for staggered, elegant fade-up effects.
* **Asymmetrical Grid:** A custom overlapping image layout in the "Why Choose Us" section to match high-end culinary editorial designs.
* **Responsive:** Fully optimized for mobile, tablet, and desktop viewing.

### 🛠️ Tech Stack
* **HTML5** * **Tailwind CSS** (via CDN for rapid, utility-first styling)
* **AOS Library** (for scroll reveals)
* **Google Fonts** (Playfair Display for headings, Inter for body text)

### 🚀 Deployment
This project is contained entirely within a single `index.html` file. Simply drop the file into your static hosting provider (GitHub Pages, Vercel, Netlify) and it is ready to go live.
